Readme · MDCopy🧮 Matrix Addition System (C++)

A console-based Matrix Addition System developed in C++ as part of Case Study 167.

This program allows users to input two matrices of the same dimensions and calculates their sum using matrix addition rules. The project demonstrates the use of 2D arrays, nested loops, functions, and input validation to perform matrix operations and display the result in a clear and formatted structure.

📌 Project Overview
Matrix addition is a mathematical operation where two matrices with the same dimensions are added element by element. Each value in the first matrix is added to the corresponding value in the second matrix to produce a result matrix.

⚙️ How the Program Works

Launch — The program starts by displaying the Main Menu
Select Option — The user selects Add Matrices
Enter Dimensions — The program asks for number of rows and columns
Input Matrix A — The user enters the elements of Matrix A
Input Matrix B — The user enters the elements of Matrix B
Calculate — The program performs element-wise matrix addition
Display Result — The result matrix is shown in formatted output
