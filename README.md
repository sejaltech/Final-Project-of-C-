🧮 Matrix Addition System (C++)

A console-based Matrix Addition System developed in C++ as part of Case Study 167.
This program allows users to input two matrices of the same dimensions and calculates their sum according to standard matrix addition rules.

The project demonstrates important programming concepts such as 2D arrays, nested loops, functions, input validation, and formatted output while implementing a fundamental mathematical operation.

📌 Project Overview

Matrix addition is a mathematical operation where two matrices of the same dimensions are added element by element.

Each value in the first matrix is added to the corresponding value in the second matrix to produce a result matrix.

Example

Matrix A

1   2
3   4

Matrix B

5   6
7   8

Result (A + B)

6   8
10  12

This project demonstrates how basic matrix operations can be implemented using core C++ programming concepts.

🎯 Objectives

The main objectives of this project are:

• Understand how matrices can be represented using 2D arrays
• Perform matrix addition using nested loops
• Practice structured programming in C++
• Implement safe input validation techniques
• Display matrices in a clear and formatted structure

✨ Features

• Menu-driven interface for easy user interaction
• Safe input validation for matrix dimensions
• Supports matrices up to 10 × 10 in size
• Displays matrices in a structured table format
• Performs Matrix A + Matrix B = Result Matrix

⚙️ How the Program Works

The program starts by displaying the Main Menu.

The user selects Add Matrices.

The program asks for the number of rows and columns.

The user enters the elements of Matrix A.

The user enters the elements of Matrix B.

The program performs element-wise matrix addition.

The result matrix is displayed in a formatted output.
